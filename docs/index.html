漢方．高級養身品牌

台興山產扎根萬華，結合早期在坪林親自養鹿的經驗，從原料源頭到製程，每一步都由家族親自把關。 「仙加味」則是為了讓傳統鹿製品，以更貼近現代人的方式被接受、被喜歡、願意長期補養。

不論是日常保養的龜鹿飲，或是深度調養的龜鹿膏，我們都相信—— 補身這件事，應該是真實看得見的用心，而不是只存在包裝上的名詞。
