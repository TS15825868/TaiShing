漢方・高級養身品牌

四代相傳，只做吃得安心的補養

來自萬華世代的鹿角世家，台興山產專注鹿與漢方養身品， 從坪林山林養鹿，到現代城市工坊，每一批龜鹿角原料都由家族親自把關。

祖父因長年專研鹿角，被親友暱稱為「鹿角」，晚輩口中敬重的「鹿角伯」； 他的名字裡也帶著「興」字，成為今日「台興」商號與紅色鹿角商標的靈感。

我們相信，補身應該兼具科學標準與老店信用。原料看得見、來源說得清， 才能讓家人每天都願意「好好補、輕鬆補」。

從龜鹿飲、龜鹿膏，到嚴選駝鹿角原料，台興山產以日常好入口的形式， 把傳統補養做得更細緻，也更貼近現代生活節奏。

仙加味 龜鹿飲

古法慢熬，每一口都是溫潤的日常補養。

補法：液態保養，建議每日 1～2 包，溫飲吸收更佳。
成分：水、鹿角、全龜、枸杞、紅棗、粉光蔘、黃耆。

仙加味 龜鹿膏

經典膏方質地，適合重視實在補養的族群。

補法：膏方濃縮，可單吃或搭配溫開水。
成分：鹿角、全龜、枸杞、紅棗、粉光蔘、黃耆。

嚴選駝鹿角原料

無腥味、好處理，專業客戶指定長年合作。

用途：漢方配方、養身食品、客製龜鹿品項。
特色：分級挑選、保存良率佳，來源透明可追溯。
