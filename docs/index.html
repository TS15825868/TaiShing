漢方．高級養身品牌

來自萬華四代的匠心傳承，台興山產專注龜鹿與漢方養身品。 從坪林山林飼鹿，到現代城市工坊，我們堅持真材實料與安心配方， 只做自己家人也願意長期補養的產品。

四代傳承，只做吃得安心的補養

台興山產的故事始於臺北萬華的街巷。曾祖父與祖父白手起家， 奔走於行口之間，也在坪林山林裡親自飼養鹿群，親手照護、親自挑選每一份原料， 只為保留最自然、最真實的補養力量。

在行口裡，祖父專門處理鹿角，同輩敬稱他為「鹿角」，晚輩則尊稱他為「鹿角伯」。 台興的「興」字，取自祖父名字中的一字，承載著把手藝興起、把品質做好、 把誠信延續的家族精神，也成為品牌最重要的根。
